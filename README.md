## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript with Apollo Federation sample repository.

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn start posts
$ yarn start users
$ yarn start todos
$ yarn start gateway

# watch mode
$ yarn dev posts
$ yarn dev users
$ yarn dev todos
$ yarn dev gateway

# debug mode
$ yarn debug posts
$ yarn debug users
$ yarn debug todos
$ yarn debug gateway

# production mode
$ yarn start:prod posts
$ yarn start:prod users
$ yarn start:prod todos
$ yarn start:prod gateway
```
